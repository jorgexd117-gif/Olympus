# Free Open-Source Agent Code References

These projects are useful if you want to extend this repository with richer multi-agent orchestration patterns.

## 1) LangGraph (MIT)

- Repo: https://github.com/langchain-ai/langgraph
- Multi-agent supervisor tutorial: https://langchain-ai.github.io/langgraph/tutorials/multi_agent/hierarchical_agent_teams/
- Why useful: graph-based orchestration, memory, and supervisor/sub-team patterns.

## 2) LangGraph Supervisor (Py)

- Repo: https://github.com/langchain-ai/langgraph-supervisor
- Why useful: ready-made supervisor handoff patterns for specialized agents.

## 3) AutoGen (MIT code, docs under CC-BY)

- Repo: https://github.com/microsoft/autogen
- Why useful: event-driven multi-agent messaging and team workflows.

## 4) CrewAI (MIT)

- Repo: https://github.com/crewAIInc/crewAI
- Org examples: https://github.com/crewAIInc/crewAI-examples
- Why useful: role-based crews and task delegation patterns.

## 5) smolagents (Apache-2.0)

- Repo: https://github.com/huggingface/smolagents
- Why useful: minimal code-agent runtime and local model support (including Ollama).
